Useful development features

Quick InstanText features overview

Collaborators

Ask other users to collaborate with you in the development of your project. Collaborators can create or modify packages and files, but they can't delete your project!

Checkpoints

Create restore points for files and projects!
This allows you to return to a previous version or browse through all checkpoints.

No Concurrency Issues

If you are coding no one else can edit your file. Collaborators will only be able to read your changes in real time.

Check Solutions

Look for solutions to your errors directly on InstanText thanks to the integrated
Google Search Bar.

Create Branch

Branch the development of the project so that you and your staff can work simultaneously on the whole project.

Merge

Return to the main branch by merging and resolving conflicts in a simple way thanks to our UI.

Fully responsive

Consult your projects on your mobile device.
